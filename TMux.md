
| Command          | Description                         |
| ---------------- |:-----------------------------------:|
| tmux new -s name | A news session with the name 'name' |

| ctrl+b+...       | Description                         |
| ---------------- |:-----------------------------------|
| %                 | Vertical split |
| "                 | Horizontal split |
| ⬆                | Move to top window |
| ⬇                | Move to bottom window  |
| ⬅                | Move to left window  |
| ➡                | Move to right window  |
| z                 | Zoom in/Zoom out  |
| c                 | Create a new tab   |
| $                 | Rename current session  |
| ,                 | Rename current tab  |
| 0..9                 | Switch between tabs  |
| w                 | Switch between tabs (choose)  |
|                  |   |
